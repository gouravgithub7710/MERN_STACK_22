BOM browser ko control karta hai, window uska main object hota hai, aur DOM window ke andar HTML page ko represent karta hai.



Event:-  JS me event ek object hota hai jo browser ke dwara diya jata hai 
          Event ka matlab hota hai koi action ya ghatna jo user ya browser ke dwara hoti hai.

          Jab bhi user kuch karta hai ya browser me kuch hota hai, usse event kehte hain.


                  | User Action    | Event       |
                  | -------------- | ----------- |
                  | Button click   | `click`     |
                  | Mouse move     | `mousemove` |
                  | Page load      | `load`      |
                  | Input me type  | `input`     |
                  | Form submit    | `submit`    |
